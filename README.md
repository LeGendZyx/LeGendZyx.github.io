# LeGendZ.github.io

庄以煊的个人网站，基于 React、Vite 和 Tailwind CSS 构建。

当前仓库名是 `LeGendZ.github.io`，所以 GitHub Pages 会作为项目页发布到 `https://legendzyx.github.io/LeGendZ.github.io/`。
如果要使用 `https://legendzyx.github.io/` 这样的个人主页短地址，仓库名需要改为 `LeGendZyx.github.io`。

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

推送到 `main` 后，GitHub Actions 会构建并部署到 GitHub Pages。
